## 👋 About

This is the source code for my personal website, jon.dev.br. It is built using TypeScript, React, Vite and Tailwind CSS.

## 🧪 Running the project locally

For this project, I'm using Vite as a development server, pre-commit with prettier and biome alongside of a latest version of Node.js.
To setup this project, please run the following commands:

    $ npm install
    $ npm run dev

## 🏗️ Building the project

    $ npm run build

## ✅ Making sure the code is fine

I'm using pre-commit in order to check if all files looks good.

    $ pre-commit run --all-files

For the CRT effect, I'm using a plug-in called; [vault66-crt-effect](https://github.com/mdombrov-33/vault66-crt-effect).
